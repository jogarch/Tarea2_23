Algoritmo para crear una clase y su instancia en JAVA

1Crear una nueva clase

Para crear una nueva clase nos vamos a nuestro proyecto damos click derecho, nuevo y seleccionamos "Java Class"

2Dar un nombre a la clase

Escribimos un nombre para nuestra clase y seleccionamos la ubicación, por defecto sera creada en la carpeta de nuestro proyecto. Para este ejemplo vamos a crear un objeto "Persona".

3 Asignar atributos y crear el constructor.

Los atributos son caracteristicas de un objeto, en este caso asignaremos 4 atributos: nombre, edad, estado civil, estatura. Un constructor es un método especial de una clase que se llama automáticamente siempre que se declara un objeto de esa clase. Su función es inicializar el objeto.

4Crear métodos

Un método es un conjunto de instrucciones definidas dentro de una clase, que realizan una determinada tarea y a las que podemos invocar mediante un nombre. Definiremos la acción de "presentarse", nos dará un mensaje en pantalla indicando todos los atributos definidos.

5Crear una instancia de clase

Una instancia es un elemento tangible (ocupa memoria durante la ejecución del programa) generado a partir de una definición de clase. Todos los objetos empleados en un programa han de pertenecer a una clase determinada.

6Acción ejecutada

En pantalla se muestra el mensaje que definimos en el método "presentarse". Finalmente Nicolai se presenta ante el mundo :)
